# Lufa

Lufa is a lightweight, efficient tool designed for network scanning and port checking within the Kali Linux environment.

## Features
* **Network Scanning:** Quickly identify active hosts on your network.
* **Port Checking:** Scan for open ports to identify potential services and vulnerabilities.
* **Lightweight:** Built with Python for ease of use and portability.

## Installation

Ensure you have Python 3 installed.

```bash
# Clone the repository
git clone https://github.com/lufaowner/Lufa.git

# Navigate to the directory
cd Lufa
```

## Usage

```bash
python3 lufa.py
```
